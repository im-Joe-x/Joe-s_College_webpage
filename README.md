# Joe College of Engineering & Technology 🏛️

A fully responsive, professional college website built with pure HTML, CSS, and JavaScript — no frameworks required.

## ✨ Features

- **Sticky Navigation** — smooth scroll with mobile hamburger menu
- **Animated Hero Section** — blueprint grid animation with live stats card and accreditation badges
- **Scrolling Marquee** — department ticker ribbon
- **About Section** — SVG campus illustration with floating achievement badge
- **Animated Counters** — scroll-triggered stats (students, alumni, placements, packages)
- **Department Cards** — 6 departments with hover gold accent bars
- **Academic Programmes** — interactive cards for all degree levels
- **Campus Life Bento Grid** — library, research centre, sports, culture, and hostels
- **Alumni Testimonials** — 3 graduate stories with star ratings
- **Admissions Section** — 4-step process + functional enquiry form with feedback
- **News & Events** — latest updates in card layout
- **Full Footer** — site map, social links, and contact details
